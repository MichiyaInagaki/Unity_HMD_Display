昭和の町モデルセット vol.3 店
"Shouwa" 80's Japanese town model set"Shouwa" 80's Japanese town model set
vol.3 -Mall


これは日本風の建築様式のモデルセットです。


当シリーズは、日本人にとって非常に親しみ深い
昭和末期(1970-1980)の特色を持つ日本の景観を表現できるモデルを提供していく予定です。

このvol.3では、個人経営店や中小企業の商業施設のモデルパーツが収録されてます。

すぐ使えるサンプルシーン、 多数のビルドエレメントと汎用マテリアルを用意してます

PC用の中・小規模の開発を想定した調整にしています。
複雑なギミックは入れず、ローポリゴン、高い汎用性、気軽に使えることを目標にしています。


商業施設は膨大で多種多様なオブジェクトが必要なため、ユニークテクスチャを少なくして汎用マテリアルを多用して、なるべくリソースを圧迫しない作り方をしました。



パッケージに含まれるモデルは、

広告塔

など


見た目を変えられる差し替え用のマテリアルを同梱しています。

建物と店と看板の種類、店内のプロップは今後のアップデートで増やしていく予定です。 

当アセットには他のシリーズのサンプルモデルも幾つか同梱しています。

-----------------------------------------

**有効な使い方**

多くの建物は、一階部分が空室になっています。shopフォルダの店パーツをはめこむことで、カスタムした建物にできます。

看板、配管、ダクト、窓、室内のパーツを追加し、壁のマテリアルを変えればオリジナルの建物が作れます。


配置した建物の間にできる隙間を違和感なく埋められる建物とパーツがいくつかあります。
密接させることで長い建物にできます。


看板のいくつかはリバーシブルになっています。少しでもマテリアル数とオブジェクト数を抑えるのが目的です。
同様に、スロープのついた敷地パーツは上下をひっくり返すことで、平面の敷地として使えます。

**使用のヒント**


* PC用の中・小規模の開発を想定した調整にしています。
* 将来は、モバイル向けに調整した町モデルをリリースまたは同梱する予定です。

*アップデートで建物、店舗、展示品、内装、看板を追加していく予定です。


* 殆どのプレハブにコライダーを実装しています。
Technie colider creatorを使って、軽くて適度に正確なコライダーを作成しました。



*プレハブはLightmap Staticを外してます。ライトの焼き込みを適用したい場合は、影をつけたいfbxのGenerate Lightmapをオンにして、プレハブのLightmap Staticをオンにしてください。


* sample Post-Processing Profile.assetはpost processing stack用の設定ファイルです。

* BTAStandardDSZF1.0.shaderは、Standard ShaderにZファイティング対策と両面表示を追加したカスタムシェーダーです。ポストエフェクトは問題なく適用できると思います。
